# 🗣️ 语音时间轴 · 智能时间记录与分析

> 不只是日历，更是你的时间可视化助手。  
> 用语音记录过去、现在与未来，让每一天都有迹可循，让时间冲突无处遁形。

## 📌 项目背景

主流日历工具和手机自带 AI（如 vivo 蓝心小V、Siri、小爱同学）主要解决“未来日程安排”，但很少有人帮你**记录过去、复盘时间**。本作品跳出传统日历思维，打造一个以**语音为核心**的**时间轴 + 智能报表 + 冲突检测 + 日程复用**系统，让你可以：

- 随时说一句“早上 8 点学语文”，事件自动上墙
- 说“现在开始写作业”，任务开始计时，结束后自动归档
- 每天/每周生成时间分配饼图，并获得 AI 改善建议
- 点击日历任意日期，查看并修改当天的日程
- 一键复用单个事件或整日安排，轻松创建周期性任务
- 自动检测时间冲突，避免双预约

## ✨ 核心功能

| 功能模块 | 说明 |
|---------|------|
| 🎤 语音录入 | 支持自然语言添加事件（未来/过去/正在进行的均可） |
| 🧠 AI 自动分类 | 自动识别“学习、工作、娱乐、生活、其他”五大类别 |
| ⏳ 正在进行任务 | 说“现在开始...”，结束时自动保存时长 |
| 📅 日历视图 | 月/周视图展示所有事件，不同类别不同颜色，支持点击查看某天 |
| 📋 选定日期时间轴 | 点击日历任意日期，右侧显示当日事件列表，支持删除、复用 |
| 🔁 事件复用 | 单个事件可复用至未来 N 天；整日安排可复制到指定日期范围 |
| ⚠️ 冲突检测 | 添加或复用时自动检测时间重叠，弹窗提示并询问是否继续 |
| 💡 智能联想 | 添加“拔智齿”等关键词事件后，AI 主动询问是否添加关联提醒（如用药） |
| 📊 智能报表 | 每日/每周生成饼图 + AI 文字建议 |
| 🔊 语音反馈 | 所有操作结果均通过语音播报，解放双手 |
| 💾 本地存储 | 数据完全保存在浏览器，无需账号，隐私安全 |

## 🛠️ 技术栈

- **纯前端**：HTML5 + CSS3 + JavaScript (ES6)
- **语音识别/合成**：Web Speech API（浏览器原生，无需额外服务）
- **大模型解析**：DeepSeek API（中文理解优秀，免费额度充足）
- **日历展示**：FullCalendar（强大的开源日历库）
- **图表绘制**：Chart.js（轻量级可视化）
- **数据存储**：localStorage（无需后端，开箱即用）

## 🚀 快速开始

### 1. 获取 DeepSeek API Key（免费）

- 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
- 注册账号 → 进入控制台 → API Keys → 创建新密钥
- 复制以 `sk-` 开头的 Key

### 2. 配置并运行

- 下载本仓库的 `index.html`
- 用记事本/VS Code 打开，找到 `const DEEPSEEK_API_KEY = "这里填你的DeepSeek API Key";`
- 替换为你的真实 Key
- 使用 **Chrome 或 Edge 浏览器** 打开 `index.html`（语音识别要求 HTTPS 或本地文件，本地文件可直接运行）
- 允许麦克风权限，点击 🎤 按钮开始体验

### 3. 语音指令示例

| 你说的话 | 会发生什么 |
|---------|------------|
| “明天下午3点开会” | 添加未来日程，自动分类为“工作” |
| “早上8点到9点学语文” | 添加过去事件，自动归为“学习” |
| “现在开始写数学题” | 开启进行中任务 |
| “结束当前任务” | 结束进行中任务并保存记录 |
| “今天做了什么” | 生成今日时间分配饼图 + AI 总结 |
| “本周报表” | 生成本周统计报表 |
| “删除包含‘开会’的事件” | 删除匹配关键词的事件 |

### 4. 手动操作（点击/按钮）

- **点击日历上的任意日期**：右侧时间轴切换显示该日期的所有事件
- **点击事件右侧「↻ 复用」**：弹出天数输入框，将当前事件复制到未来多天的同一时间段
- **点击事件右侧「🗑️ 删除」**：删除该事件
- **点击右侧顶部「复用今日全部」**：将当前选中日期的所有事件复制到指定起始日期的连续多天
- **点击「今天」按钮**：快速回到当天视图
- **点击「今日报表/本周报表」**：生成图表和语音建议
- **点击「结束当前任务」**：结束正在进行的任务

## 🌟 创新点（为何与众不同？）

1. **从“安排未来”扩展到“记录过去”**  
   填补时间管理工具空白，让每一天的真实时间分配可视化。

2. **语音优先 + 自动分类**  
   比手动打字快 3 倍，且无需用户打标签，AI 自动归类学习/工作/娱乐等。

3. **进行中任务自动计时**  
   说“现在开始...”即可开启，结束时自动计算时长，完美替代手动计时器。

4. **智能报表 + 主动建议**  
   不只展示数据，更通过 AI 生成个性化建议（如“您本周娱乐较多，适当增加学习时间”）。

5. **时间冲突检测**  
   添加或复用时自动检查重叠，避免双预约，提升日程合理性。

6. **灵活的日程复用**  
   单事件或整日一键复制，轻松应对周期性任务（如每周例会、每日运动）。

7. **智能联想**  
   根据事件关键词（拔智齿、开会等）主动询问是否需要添加关联提醒，让助手更贴心。

8. **数据私有化**  
   所有记录存在浏览器本地，不上传任何云端，保护隐私。

## 📱 如何让手机也能用？

虽然这是一个 Web 应用，但可以**像 App 一样使用**：

- **iPhone**：用 Safari 打开 → 分享按钮 → “添加到主屏幕”
- **Android**：用 Chrome 打开 → 菜单 → “添加到主屏幕”

之后就可以从手机桌面一键启动，获得接近原生 App 的体验。
